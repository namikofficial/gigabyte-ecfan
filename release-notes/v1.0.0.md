# gigabyte-ecfan v1.0.0

## Highlights
- Production-focused DKMS packaging (`dkms.conf`) for kernel update resilience.
- Post-boot-only module load strategy via systemd service template.
- Hardened install/uninstall and verification scripts.
- Initial trust baseline: security policy, troubleshooting guide, CI checks, issue/PR templates.

## Included components
- Kernel module source: `gigabyte_ec_fan.c`
- DKMS config: `dkms.conf`
- Install tooling: `scripts/install.sh`, `scripts/uninstall.sh`, `scripts/verify.sh`
- Operational docs: deployment, troubleshooting, publishing, release process

## Compatibility baseline
- Device: Gigabyte G5 MF5
- OS: Pop!_OS 24.04
- Kernel: `6.18.7-76061807-generic`
- GPU stack: NVIDIA proprietary driver

## Notes
This release is monitoring-only and prioritizes boot safety over early hardware access.
